## 🙋🏻‍♀️ 서비스 소개

🚀 Link : [my-memoPad-project](https://jiyoonz.github.io/memoPad-react-project/)

- 간단한 메모장 기능
- 메모의 제목과 내용을 작성

### 🔗 서비스 주요사항

- 메모를 작성/수정/삭제 할 수 있다.
- 즐겨찾기 메모를 따로 조회/등록할 수 있다.
- 드래그 앤 드랍을 통해 메모를 삭제할 수 있다.

## 🛠 기술스택

- **기술스택**
  - Main Language : `TypeScript`
  - Frontend  : `react` , `recoil`, `styled components` , `beautiful-dnd` ,
